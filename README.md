Project creating an earthquake map using JavaScript, Leaflet and geoJSON data. 

The final product is a earthquake map with three map styles:
1. Street
2. Satellite
3. Dark

The map also has three overlays:
1. Tectonic plates
2. Earthquakes
3. Major earthquakes
